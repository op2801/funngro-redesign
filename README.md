# Funngro Redesign

A modern redesign of the Funngro platform built using React, Vite, Tailwind CSS, and React Router.

## Features

* Modern and responsive UI
* Home Page redesign
* Teenlancers Page
* Reusable React components
* Responsive navigation
* FAQ Section
* Testimonials Section
* SEO-friendly structure
* Mobile-friendly design

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router DOM
* Lucide React

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
